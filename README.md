This is a modern calculator that has multiple functions:

• Normal arithmatic oprations (-+×÷)

• Scientific calculations (sin,cos,tan etc.)

• Unit conversions (length,temperature,speed,time etc.)

• Currency exchange (by using real time exchange rate using gemini API key)

At last, it has a history panel as well as a theme toggle option for accessible usage in any environment.

This project includes typescript, css and html.
