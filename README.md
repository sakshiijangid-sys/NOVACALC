NovaCalc is a premium computation suite that blends high-end design with AI intelligence.

🛠️ The Tech Stack
🔵 TypeScript & React 19 – Modern, type-safe architecture.
⚡ Vite – Blazing fast local development and performance.
🎨 Tailwind CSS – Custom-crafted glassmorphic styling.
🪄 Framer Motion – Fluid spring and liquid-like animations.
🧠 Gemini AI – Intelligent layer for real-time data & currency.

💎 Elite Features
🔢 Advanced Scientific Keypad – Full support for complex trigonometry and logs.
📐 7-in-1 Converter Hub – Length, Area, Mass, Temp, Data, Speed, and Time.
💸 AI Exchange Rates – Global currency rates powered by Google Gemini.
📝 Smart History Vault – Instant access and review of past sessions.

🌓 Visual Identity & Motive
🪟 Glassmorphism UI – Vibrant frosted surfaces with neon border glows.
🌙 Neon Moon Mode – A deep, focused interface for low-light work.
☀️ Ice Sun Mode – A high-contrast, professional "light" atmosphere.
🎯 The Motive – To transform static math into a vibrant, interactive experience that feels as premium as a flagship OS.
